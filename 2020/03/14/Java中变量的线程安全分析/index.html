变量的线程安全分析

本文分析对Java程序中的变量是否安全进行分析

成员变量和静态变量是否线程安全

  • 如果它们没有共享，则是线程安全的
  • 若它们被共享了，根据它们的状态是否能够改变，分成两种情况
    • 若只是读操作，则线程安全
    • 若是读写操作，则这段代码是临界区，需要考虑线程安全

局部变量是否线程安全

  • 局部变量是线程安全的
  • 但是有一种情况需要注意，当局部变量引用是对象时
    • 如果该对象没有逃离方法的作用访问，它是线程安全的
    • 如果该该对象逃离了方法的作用范围，则需要考虑线程安全
